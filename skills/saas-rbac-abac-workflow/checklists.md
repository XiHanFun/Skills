# 合并门禁清单（按阶段）

本文件聚合各角色的关键检查项，作为阶段门禁。**任一未通过不得进入下阶段**。

## 阶段 1：需求 / 设计

```
PM
- [ ] 角色清单完整（平台 + 租户）
- [ ] 权限矩阵覆盖所有资源 × 动作
- [ ] 数据范围 / 敏感字段 / ABAC 条件已明
- [ ] 验收标准可自动化
- [ ] 灰度节奏与回滚触发线写入
架构师
- [ ] 隔离模式 ADR 化
- [ ] 权限码命名空间注册
- [ ] ABAC 条件在算子白名单内
- [ ] 缓存失效路径闭环
DBA
- [ ] TenantId 策略与索引草案
- [ ] 迁移与回滚脚本草案
- [ ] 大表变更走在线 DDL
安全
- [ ] 越权三类用例已草拟
- [ ] FLS 规则草案（字段分级）
- [ ] 审计覆盖面草案
```

## 阶段 2：开发

```
架构师
- [ ] 跨模块边界与扩展点已定义
后端
- [ ] 权限码常量入仓 + Seed
- [ ] [HasPermission] 覆盖所有写入 + 敏感读
- [ ] 列表查询均走 DataScopeFilter
- [ ] 敏感字段走 FLS 管道
- [ ] TenantId 仅从 JWT / 内部可信 Header
- [ ] 权限缓存失效事件发布 + 订阅
- [ ] 审计拦截器覆盖到位
前端
- [ ] 菜单来自后端下发
- [ ] v-perm / usePerm 覆盖敏感按钮
- [ ] 路由 meta.permission 与后端一致
- [ ] 无自行脱敏逻辑
- [ ] 401/403/419 统一处理
DBA
- [ ] 所有新表含 TenantId 并建复合索引
- [ ] 唯一约束带 TenantId
- [ ] 迁移脚本含 up/down/verify
- [ ] 批量 DML 显式 WHERE TenantId
安全
- [ ] FLS 规则已配置并经审核
- [ ] 敏感操作二次确认就位
```

## 阶段 3：Code Review

```
通用
- [ ] 无硬编码权限码在前端
- [ ] 无 `if (user.TenantId != entity.TenantId)` 手写校验
- [ ] 无 `v-if=isAdmin` 判业务权限
- [ ] 无 `SELECT *` + 前端脱敏
- [ ] 无不带 WHERE TenantId 的批量 DML
后端
- [ ] 未绕过 DbContext 全局过滤器
- [ ] 权限判定未写进 AutoMapper
- [ ] 审计异步但异常已捕获
前端
- [ ] 登出 / 切租户清空 store
- [ ] URL 不携带 TenantId 参与鉴权
架构师
- [ ] 新增 ABAC 算子经评审
- [ ] 新扩展点符合白名单
```

## 阶段 4：测试

```
测试
- [ ] 每端点完成权限矩阵用例（至少含无权限/跨租户/FLS 三条）
- [ ] 隔离回归集覆盖所有租户级实体
- [ ] 审计用例覆盖成功 + 失败路径
- [ ] 性能用例含缓存失效场景
- [ ] @tenant-isolation 标签 100% 通过
- [ ] 前端 E2E 覆盖菜单/按钮/数据范围
安全
- [ ] 越权演练（水平 + 垂直 + 跨租户）通过
- [ ] FLS 四个出口（列表/详情/导出/打印）全覆盖
- [ ] 审计防篡改验证通过
DBA
- [ ] 查询计划审核：无不带 TenantId 的 seek
- [ ] 迁移脚本在 staging 真实数据量演练
PM
- [ ] 验收用例逐条签字
```

## 阶段 5：发布

```
DevOps
- [ ] 特性开关默认关闭
- [ ] 灰度计划与阶段门禁就绪
- [ ] 回滚脚本在 staging 演练通过
- [ ] 监控面板 + 告警阈值上线
- [ ] 权限 Diff 报告审批
后端
- [ ] 启动期 Permission Seed 成功
- [ ] 旧权限码下线名单确认
前端
- [ ] 版本号与后端 authMeta.version 联动
- [ ] 缓存清理策略（SW/HTTP Cache）验证
安全
- [ ] 上线后自动启动越权采样演练
PM
- [ ] 客户通知已发出
- [ ] 回滚触发线书面化
架构师
- [ ] ADR 状态更新为 "Implemented"
```

## 阶段 6：运维 / 运营

```
DevOps
- [ ] 授权失败率 / 隔离违规 / 审计写入指标健康
- [ ] 租户级面板可用
- [ ] 生产访问走堡垒机 + 审批
- [ ] 定期回滚演练（季度）
安全
- [ ] 季度越权演练
- [ ] 半年 FLS 泄露演练
- [ ] 审计日志完整性校验（哈希链）
PM
- [ ] 客户反馈闭环，Flag 生命周期维护
- [ ] 权限矩阵版本管理
架构师
- [ ] 架构债看板评审
- [ ] 废弃权限码 / Flag 清理
DBA
- [ ] 审计表归档策略执行
- [ ] 索引使用情况巡检
```

## 跨阶段红线（出现即阻断）

```
- [ ] 发现任一跨租户数据可见 → P0，立即回滚
- [ ] 发现任一字段级敏感泄露（手机号/身份证/密码明文）→ P0
- [ ] 审计链断裂 / 被篡改 → P0
- [ ] 平台超管账号被普通用户获取 → P0
- [ ] 权限变更 > 10 分钟仍未全量生效 → P1
```

## 使用方式

- 在 PR 描述中粘贴**当前阶段**的清单并逐项勾选
- CI 可通过脚本扫描 PR 描述的勾选率，< 100% 拒绝合入
- 每次上线复盘将实际执行情况与清单对比，差异入下一版
